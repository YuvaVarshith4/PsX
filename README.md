# PsX Programming Language

PsX is a lightweight experimental programming language designed with one core idea: **simplicity**. It combines the readability of Python, the flexibility of JavaScript, and the familiar syntax habits of C/C++ developers.

The language is built around **developer muscle memory**, allowing programmers to write code naturally without worrying about strict or complex syntax rules. This makes PsX easy to learn for beginners and comfortable for experienced developers.

PsX also serves as a learning platform for understanding how programming languages work internally. Its compiler/interpreter demonstrates the core stages of language execution — **lexing, parsing, and runtime execution**.

**Simple syntax. Familiar structure. Clean execution**

**PsX focuses on making programming easier to read, easier to write, and easier to understand.**
